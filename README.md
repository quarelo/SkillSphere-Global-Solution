# SkillSphere — Global Solution 2º Semestre/2025

### **Resumo do Projeto**

O SkillSphere é uma aplicação web criada como parte da Global Solution – Front-End Design – Web Development (FIAP).

Inspirado no tema "O Futuro do Trabalho – Conectando pessoas, competências e propósito por meio da tecnologia", o projeto simula uma plataforma moderna de conexão profissional (estilo LinkedIn), permitindo explorar perfis, habilidades, tecnologias e histórico dos profissionais.

A aplicação foi desenvolvida como uma SPA (Single Page Application) com experiência fluida, filtros avançados, modal de perfil completo e suporte a modo claro/escuro.

---

### **Instalação do Projeto**

**Pré-requisitos**

Node.js 18+

npm, yarn ou pnpm


**Passo a Passo**

Clone o repositório

git clone https://github.com/quarelo/SkillSphere-Global-Solution.git

cd SkillSphere-Global-Solution

Instale as dependências

npm install
ou yarn install

Execute o projeto

npm run dev

Acesse no navegador:
http://localhost:3000

## **Funcionalidades Principais**

| Funcionalidade              | Detalhes |
|-----------------------------|----------|
| Listagem de profissionais   | 60 perfis exibidos em grid responsivo |
| Busca global                | Busca por nome, cargo, cidade e tecnologia |
| Filtros avançados           | Filtros por área, cidade e tecnologia |
| Scroll horizontal           | Barra estilizada para navegabilidade |
| Modal de perfil detalhado   | About, dados pessoais/academia, experiências, habilidades técnicas, soft skills e hobbies |
| Botões de ação              | Recomendar profissional / Enviar mensagem (simulados) |
| Temas                       | Light e Dark Mode |


## **Tecnologias Utilizadas**

| Tecnologia        | Uso                                      |
|-------------------|-------------------------------------------|
| Next.js (React)   | Estrutura principal e roteamento          |
| Tailwind CSS      | Estilização e responsividade              |
| JavaScript (ES6+) | Lógica de filtros e interação             |
| JSON local        | Dados dos profissionais                   |
